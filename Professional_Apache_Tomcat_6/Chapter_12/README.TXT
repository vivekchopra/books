Chapter 12: Tomcat and IIS
--------------------------

NOTE: Before using the IIS registration script, edit it and change values
as appropriate, especially the values of
* log_file                : Path to ISAPI plugin log file
* worker_file             : Path to the 'workers.properties' worker's file
* worker_mount_file       : Path to the 'uriworkermap.properties' file

File list
---------

* isapi.reg               : Example registry settings
* conf                    : Example worker.properties and uriworkermap.properties are contained in this directory
